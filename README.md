An AI-powered code review tool that lets you paste or write code, then get intelligent feedback and suggestions in real-time.
Built using React, PrismJS, and a Node.js + Express backend powered by OpenAI.

Features:

Live code editing with syntax highlighting.
AI-generated code reviews.
Markdown-rendered output with smart formatting.
Modern UI with responsive layout.

Tech Stack:

Frontend: React, PrismJS, react-markdown, highlight.js.
Backend: Node.js, Express, OpenAI API.
Styling: CSS modules / custom styles.
